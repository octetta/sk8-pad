# `sk8-pad` - a 4x4 trigger pad for `skred`

<img src="src/sk8-pad/icon.svg" width="100">

*flexible drum machine like pads*

# quick

```
# install go1.25.5 and fyne
# go install fyne.io/tools/cmd/fyne@latest
cd src
make
./build/TARGET/sk8-pad
```
